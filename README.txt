
> This mini project was created with Cypress and Cucumber framework <

Some VSCode plugins that made your day to day easier :

- cucumber (Gherkin)
- cuke step definition generator
- vscode-icons

* In a terminal run the command below to open the cypress:

npx cypress open

* Installing project dependencies

npm install

> You will see two folders, each of them contains the .feature file that runs the corresponding tests (SwagLabs or ApiTest) <
